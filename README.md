# Loan Insights Dashboard

## Overview

The Loan Insights Dashboard is a data analysis project focused on evaluating loan application data to uncover patterns in approval decisions, risk segmentation, and customer profiles. The project demonstrates how structured data processing and visualization can support data-driven decision-making.

---

## Objectives

* Analyze trends in loan approvals and rejections
* Evaluate the impact of credit score and income on loan outcomes
* Segment applicants based on risk categories
* Develop an interactive dashboard for exploratory analysis

---

## Tools and Technologies

* Microsoft Excel – Data cleaning, transformation, and preparation
* Power BI – Data modeling, visualization, and dashboard development

---

## Dataset Description

The dataset consists of loan applicant records with the following attributes:

* Applicant income
* Loan amount
* Credit score
* Employment status
* Loan status (Approved / Rejected)
* Risk classification

---

## Data Preparation

Data preprocessing was conducted using Excel, including:

* Handling missing and inconsistent values
* Standardizing categorical variables
* Structuring data for efficient analysis
* Creating derived fields where required

---

## Dashboard Features

* KPI indicators for total applications, approvals, and rejection rate
* Loan approval distribution across credit score ranges
* Risk segmentation analysis
* Income versus loan amount comparison
* Interactive filters for dynamic exploration

---

## Key Insights

* Credit score is a primary driver of loan approval decisions
* Higher loan amounts are more likely to be rejected for lower-income applicants
* Medium-risk applicants constitute the largest proportion of the dataset
* Approval rates increase significantly within higher credit score segments
