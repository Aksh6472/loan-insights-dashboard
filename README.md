# Loan Insights Dashboard

## Overview

This project presents a comprehensive analysis of loan application data to identify key factors influencing loan approval, risk segmentation, and customer profiles. The objective is to transform raw financial data into actionable insights through structured analysis and interactive visualization.

---

## Objectives

* Analyze patterns in loan approval and rejection
* Identify relationships between credit score, income, and loan status
* Segment applicants based on risk levels
* Present insights through a clear and interactive dashboard

---

## Tools and Technologies

* **Microsoft Excel** – Data cleaning, preprocessing, and initial exploration
* **Power BI** – Data visualization and dashboard development

---

## Dataset Description

The dataset contains information on loan applicants, including:

* Applicant income
* Loan amount
* Credit score
* Employment status
* Loan status (Approved / Rejected)
* Risk category

---

## Data Preparation

Data preprocessing was performed using Excel:

* Removal of duplicates and null values
* Standardization of categorical fields
* Creation of derived columns for analysis
* Structuring data for visualization

---

## Dashboard Features

The Power BI dashboard includes:

* KPI cards highlighting total applications, approvals, and rejection rate
* Loan approval distribution by credit score
* Risk segmentation analysis
* Income vs loan amount comparison
* Interactive filters for dynamic analysis

---

## Key Insights

* Higher credit scores are strongly associated with increased loan approval rates
* Applicants with lower income levels show higher rejection rates for larger loan amounts
* Medium-risk applicants form the largest segment of loan applications
* Credit score is the most influential factor in approval decisions
